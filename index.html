Sobre o Jogo

Nosso jogo é uma experiência de sobrevivência em primeira pessoa onde o jogador assume o papel de um padre enfrentando forças demoníacas. O objetivo é derrotar os demônios que aparecem na tela digitando corretamente as palavras exibidas. A cada acerto, o inimigo é enfraquecido até ser destruído.

O jogo foi desenvolvido com HTML, CSS e JavaScript puro, com foco em mecânicas simples porém desafiadoras. Possui sprites animados que representam diferentes estados do inimigo (normal, machucado, crítico e morto), além de um sistema de pontuação para medir o desempenho do jogador.

Uso do ChatGPT

Este projeto foi desenvolvido com o auxílio do ChatGPT, que contribuiu principalmente na organização do código, sugestões de lógica para as animações e funcionamento do jogo, além de correções de erros e otimizações. Todo o conteúdo foi revisado e adaptado para garantir o aprendizado e a personalização do jogo pelos próprios desenvolvedores.
